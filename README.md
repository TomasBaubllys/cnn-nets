# CNN-Nets

The purpose of this project is to get familiar with convolutional neural networks. This Repository provides a total of 6 different networks - 3 for image clasification and 3 for timeline clasification. The code also provides different test on those models, to see how each component affects the accuracy and loss. 

## 1D CNN
To run the timeline classifier run 
```
python -m conv1dnet.conv1dMain
```
To see all the available options run
```
python -m conv1dnet.conv1dMain -h
```


## 2D CNN
To run the rock paper scissors classifier run
```
python -m conv2dnet.conv2dMain
```
To see all the available tests and options run

```
python -m conv2dnet.conv2dMain -h
```
